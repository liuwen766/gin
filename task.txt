   56 ### Request Flow
   57
   58 1. `Engine.ServeHTTP(w, req)` - http.Handler entry point
   59 2. Lazy `routeTreesUpdated.Do(updateRouteTrees)` on first request
   60 3. Context acquired from `sync.Pool`, reset, populated
   61 4. `engine.handleHTTPRequest(c)`:
   62    - Resolve path (UseEscapedPath / UseRawPath / RemoveExtraSlash)
   63    - Linear scan `engine.trees` by HTTP method
   64    - Radix tree `root.getValue()` for route matching + param extraction
   65    - On match: set handlers, `c.Next()`, `WriteHeaderNow()`
   66    - On miss: RedirectTrailingSlash → RedirectFixedPath → HandleMethodNotAl
      lowed → 404
   67 5. Context returned to pool

   69 ### Key Data Structures
   70
   71 - **`Engine`**: Embeds `RouterGroup`; holds `methodTrees` (slice of `method
      Tree`), `sync.Pool` for contexts, config flags
   72 - **`Context`**: Mutable per-request state — Request, Writer, Params, handl
      ers chain, index (int8), Keys (map), Errors, query/form cache
   73 - **`node`** (tree.go): Radix tree node — path, indices, wildChild, nType (
      static/root/param/catchAll), children, handlers, fullPath
   74 - **`responseWriter`**: Wraps `http.ResponseWriter`; tracks status, size, w
      ritten state; implements Hijacker/Flusher/CloseNotifier/Pusher
   75 - **`HandlersChain`**: `[]HandlerFunc`; `Last()` returns the terminal handl
      er

      e, Public) and Meta

   79

   82 - `skippedNodes` mechanism for backtracking during route matching
   83 - `RedirectTrailingSlash` (default true): 301/307 redirect for trailing sla

   84 - `RedirectFixedPath`: case-insensitive path correction
   85 - `HandleMethodNotAllowed`: 405 with Allow header
   86 - `RemoveExtraSlash`: normalize multiple slashes

   88 ### Binding System
   89
   90 - `Binding` interface: `Name() + Bind(*http.Request, any)`
   91 - `BindingBody`: adds `BindBody([]byte, any)`
   92 - `BindingUri`: adds `BindUri(map[string][]string, any)`
   93 - `Default(method, contentType)` dispatches: GET→Form, JSON→JSON, XML→XML,

   98 - `default` tag for default values

  架构设计

  在 tolerant/ 独立扩展包中，复用了 gin 的 binding/form_mapping.go 中的反射映射引擎，但修改了核心的 mapping() 函数，使其在遇到单字段解析错误时不立即返回，而是：                                                                                                                                                                                                                                                                                                    1. 将出错字段设置为对应类型的零值                                                                                                                                                                                                2. 收集错误明细（字段名 + 错误原因）

  ok, err := tryToSetValue(value, field, setter, tag)
